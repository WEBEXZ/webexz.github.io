Comandos para Windows

Para algunos casos es necesario contar con un dispositivo de almacenamiento con el Sistema Operativo, para poder utilizar su herramienta de reparación.

  • bootrec /fixboot
    • (Proporciona información para localizar el cargador de arranque de Windows)

  • bootrec /fixmbr
    • (Reconstruye el MBR (Master Boot Record))

  • sfc /scannow
    • (Búsqueda de errores)

  • chkdsk /f/r/x
    • (Comprueba la integridad de unidades del disco duro y repara errores lógicos en el sistema de archivos)

Protocolo HTTPS

Para utilizar el protocolo HTTPS en Apache agregar las siguientes lineas al archivo .htaccess.

  • RewriteCond %{HTTPS} !on


  • RewriteRule (.*) https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
